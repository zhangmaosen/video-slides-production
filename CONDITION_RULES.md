### 规则 A：Tesla Semi 特征（高优先级）

**触发关键词**：`["semi", "Semi", "特斯拉 Semi"]`

**描述内容**：
```
必须描述为"一辆电动半挂重型卡车"或"电动半挂重型卡车 Tesla Semi"
全封闭车身前脸（无传统进气格栅）
全宽贯穿式水平 LED 车头灯带
细长水平车头大灯无缝集成在灯带两侧
更小的环绕挡风玻璃
```

## 使用方式

**在生成脚本中**：
```python
# 1. 读取条件规则配置
rules = load_condition_rules('CONDITION_RULES.md')

# 2. 根据输入内容判断触发哪些规则
triggered_rules = check_triggers(input_content, rules)

# 3. 将触发的规则注入到元提示词模板中
final_template = inject_rules(meta_template, triggered_rules)

# 4. 调用 AI 生成提示词
prompt = generate_with_template(final_template, input_content)
```
