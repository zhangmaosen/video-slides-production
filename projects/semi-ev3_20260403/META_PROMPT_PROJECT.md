# Tesla Semi 视频幻灯片 - 元提示词模板 v2.5

## 模板目的
指导女娲为 Tesla Semi 视频生成高质量图片提示词。

---

## 核心规则

### 1. 判断画面类型
- **含文字图**：标题/副标题用「」包裹
- **通用图像**：景物/产品为主，无文字

### 2. 禁止出现的词汇
- 「必须」「严格遵守」「绝对」「禁止」「不要」「避免」
- 任何 prompt 内部指令性文字

### 3. 文字处理
- 所有画面文字用中文双引号「」
- 明确位置、字体、颜色、大小
- 若无文字：「图像中未出现任何可识别文字」

### 4. Tesla Semi 特征（记忆）
- 全封闭车身前脸，无传统进气格栅
- 全宽贯穿式水平 LED 车头灯带
- 圆润光滑的车顶线条
- 珍珠白或哑光黑色车身
- 车身颜色为纯色，无渐变

### 5. 风格参考
- 混子说风格 + 硬核工程拆解素描风格
- 故事感构图，对角线冲突

---

## 输出格式

### 正向提示词
- 风格放句首
- 空间布局（留白区域）
- 主体描述（Tesla Semi + 燃油车对比）
- 动态动作（冲击/倒塌）
- 文字渲染（用「」包裹）

### 负向提示词
```
低画质，肢体畸形，手指畸形，过饱和，蜡像感，人脸无细节
银色车身，灰色涂装，金属银，metallic silver
文字错别字，文字乱码，模糊文字
Sermi，Seemi，Semi 变形
三叉戟 logo，T 型变形
```

```

- 观点：特斯拉 Semi 掀翻燃油卡车

### 标题
- 主标题：「特斯拉 Semi 掀翻燃油卡车」
- 副标题：「一场输不起的商业货运革命」
```
